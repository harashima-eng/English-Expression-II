1. 基本の否定

+
+
+

一般動詞の否定

+

do/does/did + not を用いる。3単現のsは do に移動する点に注意。

+
+
彼は朝食を食べない。
+
He doesn't eat breakfast.
+
+
+ Step +
    +
  1. 動詞の前に do/does/did を置く
    2. +
  2. do を not で否定する（短縮可）
    3. +
  3. 動詞は原形に戻す
    4. +
+
+
+ +
+

be動詞・助動詞の否定

+

be / can / will などの直後に not を置く。疑問文と同じく語順変化なし。

+
+
その答えは正しくない。
+
The answer is not correct.
+
+
+
彼女は今夜来ないかもしれない。
+
She may not come tonight.
+
+
+ +
+

否定語の選択

+

no / none / never / nobody / nothing などは「強い否定」。文頭に置くと倒置を引き起こす場合あり。

+
+
私は二度とそこへは行かない。
+
I will never go there again.
+
+
+
+ +

2. 部分否定と数量表現

+
+
+

部分否定（not all / not every）

+

全体を否定するのではなく「一部は違う」。not all + 複数名詞 で覚える。

+
+
全ての生徒が賛成しているわけではない。
+
Not all the students agree.
+
+
+ +
+

no / little / few のニュアンス

+

no = 0、little/few = ほとんどない。a little/a few にすると「少しはある」。

+
+
その計画にはほとんど時間が残されていない。
+
There is little time left for the plan.
+
+
+ +
+

not always / not necessarily

+

頻度副詞や necessarily を否定して「いつも〜とは限らない」。

+
+
早起きが必ずしも成功を保証するとは限らない。
+
Getting up early does not necessarily guarantee success.
+
+
+
+ +

3. 否定を強調する特殊構文

+
+
+

否定語＋倒置（Hardly / Rarely / Never, etc.）

+

文頭に強い否定副詞を置くと、助動詞／do が主語より前に来る倒置が起こる。

+
+
こんなに美しい景色を見たことはほとんどない。
+
Rarely have I seen such a beautiful view.
+
+
+ 頻出の否定副詞 +
+ Hardly + Rarely + Seldom + Never + Little +
+
+
+ +
+

Not until ...

+

従属節の否定を前に出すと主節が倒置に。元の語順（It was not until ... that S V）も可。

+
+
ベルが鳴るまで、私たちは授業が終わったことに気づかなかった。
+
Not until the bell rang did we notice the class was over.
+
+
+ +
+

No sooner ... than ~

+

「〜するとすぐに…」の強調形。No sooner + had + S + pp + than で覚える。

+
+
彼女は家に着くやいなや友達に電話した。
+
No sooner had she arrived home than she called her friend.
+
+
+
+ +

4. 連結・付加の否定

+
+
+

not only A but also B（倒置あり）

+

文頭に置くと Not only + V + S ... と倒置。強調したいときに使用。

+
+
彼は英語だけでなくフランス語も話せる。
+
Not only can he speak English, but also French.
+
+
+ +
+

Neither A nor B

+

2つをともに否定。「どちらも〜ない」。動詞は直後の主語に一致させる。

+
+
私も妹も猫にアレルギーはない。
+
Neither I nor my sister is allergic to cats.
+
+
+ +
+

否定の付加（..., either / neither do I）

+

肯定に付ける too/so に対して、否定は either / neither を使用。

+
+
私は辛いものが得意ではない。彼もそうだ。
+
I am not good with spicy food. Neither am I.
+
+
+
+ +

5. 確認用クイックチェック

+
+

下線部に入る最適な形を考えて、否定構文が正しく使えているか確認しましょう。

+
    +
  1. ( ) had the train left than it began to rain. → No sooner
    2. +
  2. ( ) did she realize her mistake. → Only then / Never
    3. +
  3. He has ( ) friends in this city. → few / very few
    4. +
  4. ( ) the students passed the exam. → Not all
    5. +
  5. I can hardly wait. →「ほとんど待てない」= 我慢できないほど楽しみ、のポジティブ意味に注意
    6. +
